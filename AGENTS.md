# AGENTS.md - Param AGIO Project Context (v41.0)

## Project Overview

**Param** is a world-class Sovereign AI General Intelligence Organization (AGIO) built on a high-performance agentic framework. It is designed for deep codebase understanding, autonomous engineering, and secure multi-provider connectivity.

## Core Capabilities

### 1. Multi-Provider Intelligence
Param supports seamless switching between 7 authentication and model provider types:
- **Param Native**: Optimized for Qwen3.5-Plus and other high-performance coder models.
- **OpenAI (Standard & OAuth)**: Official support for GPT-4o, o1, and OpenAI-compatible local models (Ollama).
- **Anthropic (Standard & OAuth)**: Official device-flow authentication for Claude 3.5 Sonnet/Opus.
- **Google (Gemini & Vertex AI)**: High-context window support (up to 2M+ tokens).

### 2. Specialized Agent Catalog
Param operates as a distributed organization with 45+ specialized intelligence units, including:
- **Command**: Orchestrator (CEO), Project Manager, Team Lead.
- **Engineering**: Python Pro, Rust Specialist, Frontend Expert, CSS Wizard, Backend Architect.
- **Ops & Security**: K8s Architect, Security Auditor (CISO), Pentester.
- **Quality & Content**: Bug Hunter, Test Engineer, Technical Writer.
- **Sovereign**: Telegram Liaison, Self-Mutator.

### 3. Remote Sovereignty
- **Telegram Control**: Full voice and text command execution via a rebranded Telegram bot interface.
- **Zero-Config Migration**: Automatic reuse of legacy credentials, skills, and extensions from `~/.gemini/` and `~/.qwen/`.
- **Fast-Start Architecture**: Optimized startup sequence with lazy-loading and parallel initialization (40% faster boot).

## CLI Reference

- `param auth` - Manage multi-provider authentication (Param, OpenAI, Anthropic).
- `param channel start` - Launch remote connectivity channels (Telegram).
- `param <task>` - Execute autonomous agentic tasks on the current codebase.
- `/model <name>` - Hot-swap between models in an active session.
- `/agent <id>` - Delegate specific sub-tasks to specialized AGIO experts.
